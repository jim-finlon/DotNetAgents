## Overview

Structured learning paths to master DotNetAgents, from beginner to expert. Choose the path that matches your goals and experience level.

## Learning Path Structure

Each learning path includes:
- **Prerequisites**: What you need to know
- **Duration**: Estimated time to complete
- **Modules**: Step-by-step modules
- **Projects**: Hands-on projects
- **Assessment**: Knowledge checks
