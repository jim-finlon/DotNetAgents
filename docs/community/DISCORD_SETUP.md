## Roles

### User Roles

1. **@everyone** - Default role
   - Read-only access to public channels
   - Can post in general channels

2. **@Member** - Verified member
   - Full access to community channels
   - Can create threads

3. **@Contributor** - Active contributor
   - Access to contributor channels
   - Can moderate discussions
   - Special badge

4. **@Maintainer** - Project maintainer
   - Full server access
   - Can manage channels
   - Can assign roles

### Special Roles

- **@New Contributor** - First-time contributors
- **@Documentation** - Documentation contributors
- **@Translation** - Translation contributors

## Setup Steps

### 1. Create Server

1. Open Discord
2. Click "+" to create server
3. Choose "Create My Own"
4. Name: "DotNetAgents Community"
5. Upload server icon

### 2. Create Channels

Create channels according to the structure above:
- Use categories for organization
- Set appropriate permissions
- Add channel descriptions

### 3. Configure Roles

1. Create roles as defined above
2. Set role colors
3. Configure permissions
4. Set role hierarchy

### 4. Set Up Bots

1. Invite GitHub bot
2. Configure webhooks
3. Set up moderation bot
4. Configure welcome bot

### 5. Create Rules Channel

Create `#rules` channel with:
- Code of Conduct
- Community guidelines
- Channel purposes
- Role explanations

### 6. Set Up Auto-Moderation

1. Enable auto-moderation
2. Configure filters
3. Set up warnings
4. Configure timeouts

## Moderation Guidelines

### Moderation Principles

1. **Be Fair**: Apply rules consistently
2. **Be Transparent**: Explain actions
3. **Be Helpful**: Guide, don't just punish
4. **Be Respectful**: Treat everyone with dignity

### Common Actions

- **Warning**: First-time minor violations
- **Timeout**: Temporary mute (1 hour - 1 day)
- **Kick**: Remove from server (can rejoin)
- **Ban**: Permanent removal (serious violations)
