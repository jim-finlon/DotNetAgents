### Getting Involved

1. **Join the Community**
   - Join Discord server
   - Follow on GitHub
   - Subscribe to newsletter

2. **Start Contributing**
   - Read [Contributing Guide](../../CONTRIBUTING.md)
   - Find good first issues
   - Make your first contribution

3. **Share Your Work**
   - Submit to showcase
   - Share in Discord
   - Write blog posts
